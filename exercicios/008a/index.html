Texto marcado

Podemos criar também um texto marcado usando a tag MARK

Para alterar a cor do marcador, incluimos dentro da tag mark o parâmetro style, e dentro do style selecionamos background-color

Caso você precise alterar a cor para ínumeros mark, é mais fácil incluir a tag style no head, fazendo um seletor para mark da mesma forma acima
